# 🚀 SpaceXplore – Space Exploration Landing Page

A modern, responsive, and visually stunning landing page built to showcase the wonders and importance of **space exploration**. Designed with clean HTML, CSS, and a touch of JavaScript, this project is perfect for learning and demonstrating frontend fundamentals.

## 🌌 Project Overview

This landing page includes:

* A sticky navigation bar
* Eye-catching hero section
* Feature cards that explain the benefits of space exploration
* A call-to-action button
* Fully responsive design

## 🛠️ Tech Stack

* HTML
* CSS
* Vanilla JavaScript (used minimally for interactivity)

## 🎨 Features

* 💫 Cosmic gradient background and glowing text
* 🔥 Interactive "Join Our Space Journey" button
* 🌟 Three key benefit cards: Innovation, Discovery, and Inspiration
* 📱 Mobile-friendly design with responsive breakpoints
* ♿ Accessibility with ARIA labels and keyboard focusability

## 📌 Purpose

Built as part of a development assignment/task with the goal to:

* Create a themed landing page using HTML/CSS
* Focus on design, responsiveness, and accessibility
* Deploy the site using GitHub Pages
